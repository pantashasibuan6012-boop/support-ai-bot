# Support AI Bot

AI-powered customer support automation with ticket classification, sentiment analysis, and automated response generation.

## Features
- Multi-channel ticket ingestion (email, Slack, web)
- AI-powered ticket classification & priority scoring
- Sentiment analysis for customer messages
- Auto-response generation with confidence scoring
- Escalation rules with human handoff

## Installation
```
pip install -r requirements.txt
```

## Usage
```
python main.py classify "My order hasn't arrived yet"
python main.py serve --port 8080
```
